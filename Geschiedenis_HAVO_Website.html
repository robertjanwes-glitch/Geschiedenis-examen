Voer je Worker-URL in
Deze tool gebruikt een Cloudflare Worker als proxy voor de AI. Vul hieronder de URL van jouw Worker in om te beginnen.
Nog geen Worker? Zie de instructies hieronder — gratis, 5 minuten.
Worker instellen (eenmalig, gratis)
1
Ga naar workers.cloudflare.com → maak gratis account → klik Create Worker
2
Vervang de standaard code door de meegeleverde worker.js en klik Deploy
3
Ga naar Settings → Variables en voeg toe:
ANTHROPIC_API_KEY = sk-ant-...
4
Kopieer de Worker-URL (bijv. mijn-worker.xyz.workers.dev) en plak hem hierboven
